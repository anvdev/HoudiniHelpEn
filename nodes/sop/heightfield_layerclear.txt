#type:     node
#context:  sop

"""Sets all values in a heightfield layer to a fixed value."""

@parameters

Number of clears:
    How many layers to clear.

Layer #:
    #id: layer#
    
    The name of the layer to clear.
	
Value
    #id: value#
	
	The value which is used to clear the layer.
