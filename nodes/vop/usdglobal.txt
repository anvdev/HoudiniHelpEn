= USD Global Variables =

#type: node
#context: vop
#internal: usdglobal
#icon: VOP/usdglobal

"""Provides outputs representing commonly used input variables for processing USD primitive attributes inside an Attribute VOP LOP."""

This operator provides outputs that represent all the global variables in an Attribute VOP LOP network. There are no inputs to this operator.

Most of the parameters provided refer to Universal Scene Description (USD) primitive metadata.

@parameters

== Inputs ==

None.

== Outputs ==

Primitive Path:
    Full path of the USD primitive.


@related

- [Node:lop/attribvop]
