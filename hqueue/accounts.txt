= The "hquser" Account =
The 'hquser' account can be optionally created when adding a client machine to the farm.

On Linux machines, the account has the following properties:

Property ||
    Value ||

username |
    hquser
password |
    hqpass
id |
    601
home directory | 
    /home/hquser
group | 
    hqgroup

In addition to the "hquser" account, a new group is also created with the group name set to "hqgroup" and the group id set to 601.
