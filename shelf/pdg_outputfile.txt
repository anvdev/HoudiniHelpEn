= TOPs Preview Output =

"""Creates a file node to read the output for the current active PDG work item"""

== Using TOPs Preview Output == (includeme)
    
    # Click the [Icon:SHELF/pdg_outputfile] __Preview Output__  tool on the __TOPs Houdini Tools__ tab.
    
    # A geo with a file node will be created to read `@pdg_output`.

This will result in the output geo (if there is any) of the selected work item being loaded.
