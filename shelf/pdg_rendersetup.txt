== Using TOPs Render Setup == (includeme)
    
    # Click the [Icon:SHELF/pdg_rendersetup] __Render Setup__  tool on the __TOPs Tools__ tab.
    
    # Choose whether to generate an IFD before rendering.
    
    # Select the camera to render. If no camera is selected the default camera will be used.
    
If generate IFD is selected, `generateifd` and `renderifd` nodes will be created, otherwise a `ropmantra` render node is created.
