= Straighten Guides =

"""Straightens the hair by bending each segment back so that it's in the same direction as the previous segment. """

This tool straightens the hair by bending each segment back so that it's in the same direction as the previous segment. This effectively removes any bending, frizz, etc. that was previously applied.

Clicking [Icon:FUR/paint_straighten] __Straighten Guides__ on the __Guide Process__ tab will put down a [Guide Process SOP|Node:sop/guideprocess] with the __Operation__ parameter set to __Straighten__. For more information, see the [Guide Process SOP|Node:sop/guideprocess] help page.
