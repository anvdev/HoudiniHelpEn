= Frizz Guides =

"""Offsets the curve points along the guide hairs to create a frizzy look."""

This tool offsets curve points along the guide hairs using noise, varying along the curve's length and different for each curve.

Clicking [Icon:FUR/paint_frizz] __Frizz Guides__ on the __Guide Process__ tab will put down a [Guide Process SOP|Node:sop/guideprocess] with the __Operation__ parameter set to __Frizz__.  For more information, see the [Guide Process SOP|Node:sop/guideprocess] help page.
